# Images for MyFitnessAppV2

Images are copied from the original [MyFitnessApp](https://github.com/GeorgiGradev/MyFitnessApp) repo.

## How to copy images

1. Clone or download the original repo:
   - **Repo:** https://github.com/GeorgiGradev/MyFitnessApp
   - **Path:** `MyFitnessApp/Web/MyFitnessApp.Web/wwwroot/images`

2. Copy the contents of that `images` folder into this folder (`ClientApp/public/images/`), preserving subfolders:
   - `articlecategories/`
   - `articles/`
   - `exercisecategories/`
   - `exercises/`
   - `homepage/` (e.g. mainimage01.jpg, mainimage02.jpg)
   - `indexpage/`
   - `logo/` (e.g. logo01.jpgâ€“logo05.jpg)
   - `profileimages/`

3. The app uses:
   - **Logo:** `/images/logo/logo01.jpg` (header and footer)
   - **Home page:** `/images/homepage/mainimage01.jpg` and `mainimage02.jpg` (Welcome section; shown side by side)
   - **Index page (optional):** `/images/indexpage/` (e.g. anatomy.png or IndexPage01.jpg) if you use a single main image

If images are not copied yet, the UI shows text/placeholders and uses `alt` text for accessibility.
